# 设置对话框 (Settings Dialog)

## 功能概述

设置对话框用于配置EEG仿真器的全局参数，包括语言、主题、日志级别、默认仿真参数和滤波器阶数等。

**打开方式**：菜单栏 → Project → Settings

## 界面布局

设置按功能分组显示：
- **基础设置**：语言、主题、日志级别
- **仿真设置**：默认采样率、刷新率、滤波阶数
- **项目设置**：默认项目目录

## 参数详解

### 1. 语言设置

**参数**：`Language`

**选项**：
- **中文 (简体)**：简体中文界面
- **English**：英文界面

**效果**：立即应用，无需重启

**影响范围**：
- 所有菜单和标签
- 提示信息
- 文档和工具提示

---

### 2. 主题设置

**参数**：`Theme`

**选项**：
- **Dark Mode**：深色主题（推荐，护眼）
- **Light Mode**：浅色主题

**效果**：立即应用，无需重启

**深色主题特点**：
- 深灰/黑色背景
- 高对比度绿色/蓝色强调
- 适合长时间使用

---

### 3. 日志级别

**参数**：`Log Level`

**选项**：
| 级别 | 说明 | 用途 |
|------|------|------|
| Debug | 所有信息，包括详细调试 | 开发和故障排除 |
| Info | 一般信息 | 正常运行监测 |
| Warning | 警告及以上 | 仅关注潜在问题 |
| Error | 仅错误信息 | 最小日志量 |

**日志位置**：`logs/`目录下，按日期命名

**用途**：
- 追踪程序运行状态
- 诊断问题
- 性能分析

---

### 4. 默认采样率

**参数**：`Default Sampling Rate` (单位：Hz)

**范围**：100 - 10000 Hz

**默认值**：1000 Hz

**意义**：
- 新项目的默认采样率
- 影响时间精度：dt = 1/rate

**推荐值**：
- 一般研究：1000 Hz
- 高频分析：2000-5000 Hz
- 慢波研究：250-500 Hz

---

### 5. 热力图刷新率

**参数**：`Heatmap Refresh Rate` (单位：ms)

**范围**：50 - 5000 ms

**默认值**：1000 ms

**意义**：
- Signal Page中热力图的更新间隔
- 较短的间隔 = 更实时但更多CPU占用

**推荐值**：
- 实时监测：100-500 ms
- 一般使用：1000 ms
- 长时记录：2000-5000 ms

---

### 6. 滤波阶数

滤波器阶数决定滤波的陡峭程度：

#### 6.1 高通阶数

**参数**：`HP Order`

**范围**：1 - 10

**默认值**：4

**意义**：
- 阶数越高，截止越陡峭
- 但相位失真和延迟也越大

**效果**：
| 阶数 | 滚降 | 延迟 | 用途 |
|------|------|------|------|
| 2 | 12dB/oct | 小 | 简单滤波 |
| 4 | 24dB/oct | 中 | 推荐，平衡 |
| 6 | 36dB/oct | 大 | 强衰减需求 |
| 8+ | 48dB+/oct | 很大 | 特殊需求 |

---

#### 6.2 低通阶数

**参数**：`LP Order`

**范围**：1 - 10

**默认值**：4

**意义**：同高通，用于低通滤波

---

#### 6.3 陷波阶数

**参数**：`Notch Order`

**范围**：1 - 10

**默认值**：2

**意义**：
- 转换为Q值：Q = 15 × order
- 决定陷波带宽

**Q值与带宽**：
```
带宽 = 中心频率 / Q

例：50Hz陷波，Q=30
带宽 = 50/30 ≈ 1.67Hz
```

**推荐值**：
| 阶数 | Q值 | 带宽(50Hz) | 用途 |
|------|-----|-----------|------|
| 1 | 15 | 3.3Hz | 宽陷波 |
| 2 | 30 | 1.7Hz | 推荐 |
| 4 | 60 | 0.8Hz | 窄陷波 |
| 6+ | 90+ | <0.6Hz | 超窄（可能不稳定）|

---

### 7. 默认项目目录

**参数**：`Default Project Directory`

**功能**：设置新建项目的默认保存位置

**设置方法**：
1. 点击"Browse..."按钮
2. 在文件浏览器中选择目录
3. 点击"OK"确认

**建议**：
- 使用专用工作目录
- 确保有足够磁盘空间
- 使用SSD提高性能

---

## 操作按钮

### Save（保存）

**功能**：保存所有设置更改

**效果**：
- 写入配置文件
- 立即应用可热更新的设置
- 关闭对话框

### Cancel（取消）

**功能**：放弃所有更改

**效果**：
- 不保存任何修改
- 关闭对话框
- 保持原有设置

---

## 配置文件

设置保存在：`~/.eegs/config.db`（SQLite 数据库）

**示例内容**：
```json
{
    "language": "zh_CN",
    "theme": "dark",
    "log_level": "DEBUG",
    "default_sampling_rate": 1000,
    "heatmap_refresh_interval": 1000,
    "filter_highpass_order": 4,
    "filter_lowpass_order": 4,
    "filter_notch_order": 2,
    "default_project_dir": "/Users/xxx/EEG_Data"
}
```

---

## 使用建议

### 首次使用

1. 设置默认项目目录
2. 选择熟悉的语言
3. 选择偏好的主题
4. 保持其他默认参数

### 性能优化

如果电脑配置较低：
- 降低热力图刷新率（2000-5000ms）
- 使用较低滤波阶数（2-4）
- 减少日志级别（Warning或Error）

### 研究用途

如果需要精确分析：
- 适当提高滤波阶数（4-6）
- 使用Debug日志级别记录详细信息
- 设置专用的数据目录

---

## 故障排除

| 问题 | 可能原因 | 解决方案 |
|------|----------|----------|
| 设置不保存 | 目录权限问题 | 检查config目录权限 |
| 语言不切换 | 需要重启 | 重启程序 |
| 主题不切换 | 显示缓存 | 最小化再最大化窗口 |
| 设置丢失 | 配置文件损坏 | 删除config.json重新设置 |

## 相关概念

### 滤波器阶数详解

Butterworth滤波器的滚降特性：
```
滚降速率 = 6 × 阶数 dB/octave

4阶：24dB/octave
- 频率加倍（如10→20Hz），衰减24dB
- 10倍频程（如10→100Hz），衰减40dB
```

### 群延迟

滤波器引入的时间延迟：
```
延迟 ≈ 阶数 / (2 × 截止频率)

例：4阶 @ 1Hz ≈ 2秒延迟
```

这是显示延迟，不影响保存的原始数据。

### 陷波Q值

品质因数Q的定义：
```
Q = 中心频率 / -3dB带宽

高Q = 窄带 = 更精确但可能不稳定
低Q = 宽带 = 更稳定但影响邻近频率
```
