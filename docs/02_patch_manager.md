# Patch管理器 (Patch Manager)

## 功能概述

Patch管理器用于创建和管理基于Label的偶极子组（Patch）。Patch是一组空间上相邻、共享相同波形设置的偶极子集合，简化了大规模偶极子仿真的管理。

## 界面布局

界面分为三个面板：
- **左侧面板**：MRI切片可视化 + 所有偶极子位置
- **中间面板**：Patch列表和编辑区
- **右侧面板**：Label表格（左/右脑分区）

## 参数详解

### 1. Patch列表

**功能**：显示所有已创建的Patch

**操作**：
- 单击Patch：编辑该Patch
- 右键菜单：删除、重命名

**显示信息**：
- Patch名称
- 包含的偶极子数量
- 波形类型图标

---

### 2. Patch基本信息

#### 2.1 Patch名称
**参数**：`Patch Name`

**功能**：设置Patch的标识名称

**命名建议**：
- 包含位置信息：`lh-superiortemporal-v1234`
- 包含功能信息：`auditory-cortex-left`

---

#### 2.2 中心偶极子
**参数**：`Center Dipole`

**功能**：显示当前选中的中心偶极子

**意义**：Patch以中心偶极子为圆心，半径内的所有源点都会包含在Patch中

---

#### 2.3 半径设置
**参数**：`Radius` (单位：mm)

**范围**：1-100 mm

**默认值**：5.0 mm

**功能**：定义Patch的空间范围

**意义**：
- 半径越大，包含的偶极子越多
- 半径内的所有源点会继承相同的波形设置
- 模拟局部脑区的同步活动

**显示**：`Affected Dipoles` 显示半径内将被影响的偶极子数量

---

### 3. 波形设置

#### 3.1 波形类型
**参数**：`Waveform Type`

**选项**：
| 类型 | 描述 | 用途 |
|------|------|------|
| Sin | 正弦波 | 稳态节律（如α波10Hz） |
| Cos | 余弦波 | 相位偏移的正弦活动 |
| ERP | 事件相关电位 | 瞬态响应（如P300） |
| Gaussian | 高斯调制 | 短时爆发活动 |
| Gamma | 伽马函数 | 快速瞬态响应 |
| Oscillation | 高斯包络振荡 | 瞬时节律爆发 |
| Custom | 自定义数据 | 导入外部波形 |

---

#### 3.2 正弦波参数 (Sin/Cos)

| 参数 | 单位 | 默认值 | 说明 |
|------|------|--------|------|
| Amplitude | nAm | 10.0 | 峰值振幅（纳安米） |
| Frequency | Hz | 10.0 | 振荡频率 |
| Phase | ° | 0 | 初始相位（度） |
| Offset | - | 0 | 直流偏置 |
| Onset | s | 0 | 开始时间（秒） |
| Duration | s | 0 | 持续时间（0=无限） |

**振幅意义**：
- 典型EEG信号：10-100 nAm
- 强事件相关：100-500 nAm
- 癫痫样活动：>1000 nAm

---

#### 3.3 ERP参数

| 参数 | 单位 | 默认值 | 说明 |
|------|------|--------|------|
| Amplitude | nAm | 10.0 | 峰值振幅 |
| Frequency | Hz | 1.0 | 载波频率（用于振荡ERP） |
| Latency | s | 0.1 | 峰值延迟时间 |
| Width | s | 0.05 | 高斯包络宽度 |
| Polarity | - | positive | 正/负极性 |

**延迟意义**：
- P50: ~50ms
- N100: ~100ms
- P300: ~300ms

---

#### 3.4 高斯参数

| 参数 | 单位 | 默认值 | 说明 |
|------|------|--------|------|
| Sigma | s | 0.1 | 高斯函数标准差 |
| Center | 0-1 | 0.5 | 脉冲中心位置（相对时间） |

---

#### 3.5 振荡参数 (Oscillation)

| 参数 | 单位 | 默认值 | 说明 |
|------|------|--------|------|
| Frequency | Hz | 10.0 | 振荡频率 |
| Center | s | 0.5 | 包络中心时间 |
| Width | s | 0.1 | 包络宽度 |
| Amplitude | nAm | 20.0 | 峰值振幅 |

**用途**：模拟瞬时节律爆发，如gamma burst

---

### 4. MNE幅度因子

**参数**：`Amplitude Scale`

**范围**：1e-12 ~ 1e-6

**默认值**：1e-9

**功能**：MNE源信号幅度的全局缩放因子

**物理意义**：
- 默认值1e-9对应约10nAm的生理合理范围
- 用于调节Patch内所有偶极子的整体强度
- 与单个波形振幅相乘得到最终源强度

**公式**：
```
最终源强度 = 波形振幅 × 幅度因子
```

---

### 5. 波形预览

**功能**：实时预览当前波形设置的效果

**显示**：
- X轴：时间（秒）
- Y轴：振幅（nAm）
- 自动调整显示范围

**用途**：在保存前验证波形参数是否正确

---

## 使用流程

### 创建新Patch

```
1. 在右侧Label表格选择目标脑区
   ↓
2. 在中间"源点列表"点击选择中心偶极子
   ↓
3. 设置Radius（半径）
   ↓
4. 查看"Affected Dipoles"确认数量
   ↓
5. 选择Waveform Type
   ↓
6. 设置波形参数
   ↓
7. 在预览窗口确认波形
   ↓
8. 点击"Save Patch"
```

### 编辑现有Patch

```
1. 在左侧Patch列表选择目标Patch
   ↓
2. 修改波形参数或半径
   ↓
3. 点击"Save Patch"保存更改
```

## 相关概念

### 什么是Patch？

Patch是空间上相邻的偶极子组：
- 基于距离（半径）自动包含邻近源点
- 所有成员共享相同的波形设置
- 模拟局部脑区的同步神经活动

### Patch vs 单个偶极子

| 特性 | Patch | 单个偶极子 |
|------|-------|-----------|
| 空间范围 | 局部区域（多源点） | 单点 |
| 波形设置 | 统一设置 | 独立设置 |
| 适用场景 | 扩展性脑区活动 | 局灶性活动 |
| 计算效率 | 高效（批量管理） | 繁琐（逐个设置） |

### 半径选择的考虑

- **小半径（1-3mm）**：局灶性活动，如局灶性癫痫
- **中半径（5-10mm）**：功能区活动，如听觉皮层
- **大半径（>10mm）**：广泛性脑区，如默认模式网络
