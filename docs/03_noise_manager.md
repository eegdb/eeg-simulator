# 噪声管理器 (Noise Manager)

## 功能概述

噪声管理器用于配置和添加多种类型的噪声到仿真信号中，模拟真实EEG记录中的各种干扰源，提高仿真的真实性。

## 界面布局

- **左侧**：噪声类型选择列表
- **右侧**：选中噪声类型的参数配置

## 噪声类型详解

### 1. 高斯白噪声 (Gaussian White Noise)

**描述**：最基本的随机噪声，频谱均匀分布

**参数**：

| 参数 | 单位 | 默认值 | 说明 |
|------|------|--------|------|
| Amplitude | μV | 1.0 | 噪声标准差（RMS值） |

**物理意义**：
- 模拟放大器电子噪声
- 模拟背景脑活动（非相干部分）
- 典型EEG系统噪声：1-5 μV RMS

**频谱特性**：所有频率分量功率相等

---

### 2. 粉红噪声 (Pink Noise / 1/f Noise)

**描述**：功率随频率降低的噪声，符合自然信号特性

**参数**：

| 参数 | 单位 | 默认值 | 说明 |
|------|------|--------|------|
| Amplitude | μV | 1.0 | 噪声幅度 |

**物理意义**：
- 模拟背景脑电活动（1/f特性）
- 模拟生物信号的自然频谱分布
- 真实EEG背景噪声的主要成分

**频谱特性**：功率 ∝ 1/f

---

### 3. 工频干扰 (Power Line Interference)

**描述**：来自电网的周期性干扰

**参数**：

| 参数 | 单位 | 默认值 | 说明 |
|------|------|--------|------|
| Amplitude | μV | 2.0 | 干扰幅度 |
| Frequency | Hz | 50 | 工频（中国/欧洲50Hz，美国60Hz） |

**物理意义**：
- 来自电源线的电磁干扰
- 通过不良接地或电极引入
- 真实EEG中最常见的干扰源

**抑制方法**：
- 在Signal Page中启用Notch滤波
- 推荐设置：Q=30-150

---

### 4. 基线漂移 (Baseline Drift)

**描述**：缓慢的低频波动

**参数**：

| 参数 | 单位 | 默认值 | 说明 |
|------|------|--------|------|
| Amplitude | μV | 5.0 | 漂移幅度 |
| Frequency | Hz | 0.5 | 漂移频率（通常<1Hz） |

**物理意义**：
- 电极极化电位变化
- 皮肤电位缓慢变化
- 受试者轻微运动

**抑制方法**：
- 高通滤波（>0.5Hz或>1Hz）
- 在Signal Page中设置Highpass

---

### 5. 脉冲噪声 (Impulse Noise)

**描述**：随机出现的短时高幅脉冲

**参数**：

| 参数 | 单位 | 默认值 | 说明 |
|------|------|--------|------|
| Amplitude | μV | 50.0 | 脉冲峰值幅度 |
| Probability | 0-1 | 0.001 | 每采样点出现概率 |

**物理意义**：
- 肌肉抽搐（EMG）
- 电极接触不良
- 外部电磁脉冲

---

### 6. 通道特定噪声

**描述**：为特定通道单独添加的噪声

**参数**：

| 参数 | 说明 |
|------|------|
| Channel | 目标通道名称 |
| Noise Type | 上述任意噪声类型 |

**用途**：
- 模拟特定通道的坏电极
- 模拟局部干扰
- 测试通道选择算法的鲁棒性

---

## 噪声叠加原理

所有配置的噪声会**线性叠加**到信号中：

```
最终信号 = 仿真信号 + Σ(各类型噪声)
```

> **技术说明（待改进）**  
> 当前版本中，每种噪声在**各通道独立生成**（无空间相关）。这便于实现与时间连续性，但与真实 EEG 中 EOG/工频/ECG 等高度通道相关的特性不符，也可能使多通道波形看起来「相位错位」。  
> 详见：[noise_spatial_model_todo.md](noise_spatial_model_todo.md)

**注意事项**：
- 噪声幅度可以超过信号幅度（模拟低信噪比情况）
- 过多噪声会淹没真实信号，降低仿真质量
- 建议逐步添加噪声，观察对信号的影响

## 使用流程

```
1. 点击"Add Noise"添加噪声源
   ↓
2. 选择噪声类型
   ↓
3. 设置噪声参数
   ↓
4. 预览噪声效果（可选）
   ↓
5. 点击"OK"保存
   ↓
6. 重复1-5添加更多噪声源
   ↓
7. 点击"Apply"应用所有噪声
```

## 推荐配置

### 高质量仿真（低噪声）

| 噪声类型 | 幅度 | 用途 |
|----------|------|------|
| 粉红噪声 | 1 μV | 背景脑活动 |
| 工频干扰 | 0.5 μV | 轻微电源干扰 |

**总噪声水平**：约2 μV RMS

### 真实环境仿真（中等噪声）

| 噪声类型 | 幅度 | 用途 |
|----------|------|------|
| 粉红噪声 | 3 μV | 背景脑活动 |
| 工频干扰 | 2 μV | 电源干扰 |
| 基线漂移 | 5 μV @ 0.3Hz | 电极漂移 |

**总噪声水平**：约5-8 μV RMS

### 恶劣环境仿真（高噪声）

| 噪声类型 | 幅度 | 用途 |
|----------|------|------|
| 粉红噪声 | 5 μV | 背景脑活动 |
| 工频干扰 | 5 μV | 强电源干扰 |
| 基线漂移 | 10 μV | 严重电极问题 |
| 脉冲噪声 | 100 μV | 肌肉抽搐 |

**总噪声水平**：>15 μV RMS

## 噪声管理策略

### 1. 渐进式添加

从低噪声开始，逐步增加，观察：
- 信号特征是否仍可识别
- 波形检测算法的性能变化
- 频谱分析的准确性

### 2. 分频段分析

使用Signal Page的FFT功能观察噪声频谱：
- 低频段（<4Hz）：检查基线漂移
- 50/60Hz：检查工频干扰
- 高频段（>100Hz）：检查高频噪声

### 3. 与滤波配合

在Signal Page设置滤波参数抑制噪声：
- 高通：抑制基线漂移
- 低通：抑制高频噪声
- 陷波：抑制工频干扰

## 噪声频谱特性详解

### 频谱特性总览

| 噪声类型 | 功率谱密度 | 频谱特征 | 生成方法 |
|---------|-----------|---------|---------|
| **白噪声** | $P(f) = C$ | 平坦频谱 | 纯随机序列 |
| **粉红噪声** | $P(f) = C/f$ | 1/f 衰减 | 白噪声积分 |
| **布朗噪声** | $P(f) = C/f^2$ | 1/f² 衰减 | 白噪声双重积分 |
| **1/f 噪声** | $P(f) = C/f^\alpha$ | α 可调衰减 | 分数积分滤波 |

### 白噪声 (White Noise)

**频谱特征**：**平坦** (Flat Spectrum)，所有频率成分能量相同

| 属性 | 说明 |
|------|------|
| **物理含义** | 类似白光包含所有颜色，各频率能量相等 |
| **听起来** | 嘶嘶声，类似电视雪花 |
| **EEG 意义** | 模拟电子热噪声、量化噪声 |
| **数学表达** | 随机数序列，各样本独立不相关 |

**FFT 图像**：近似水平直线（各频率幅度相近）

**代码实现**：
```python
noise = amplitude * np.random.randn(n_samples)
```

---

### 粉红噪声 (Pink Noise)

**频谱特征**：**1/f 衰减**，每倍频程能量相等

| 属性 | 说明 |
|------|------|
| **物理含义** | 低频能量高，高频能量低，能量按 1/f 衰减 |
| **听起来** | 更"柔和"，类似风声或流水 |
| **EEG 意义** | 接近真实脑电背景活动（脑电具有 1/f 特性）|

**能量分布**：
- 1-2 Hz: 能量 = 1
- 2-4 Hz: 能量 = 0.5（总和与 1-2Hz 相同）
- 4-8 Hz: 能量 = 0.25

**FFT 图像**：从左到右单调下降（斜率约 -10 dB/倍频程）

**代码实现**（近似）：
```python
white = np.random.randn(n_samples)
pink = np.cumsum(white)  # 积分 → 1/f 特性
pink = pink / np.std(pink) * amplitude
```

---

### 布朗噪声 (Brown Noise / Red Noise)

**频谱特征**：**1/f² 衰减**，低频占绝对主导

| 属性 | 说明 |
|------|------|
| **物理含义** | 比粉红噪声衰减更快，随机游走特性 |
| **听起来** | 低沉的隆隆声 |
| **EEG 意义** | 模拟电极极化漂移、缓慢基线漂移 |

**与白噪声对比**：
- 白噪声：相邻样本独立无关
- 布朗噪声：强相关性（随机游走，当前值 = 前值 + 随机步长）

**FFT 图像**：急剧下降（斜率约 -20 dB/倍频程）

**代码实现**：
```python
white = np.random.randn(n_samples)
brown = np.cumsum(np.cumsum(white))  # 双重积分 → 1/f²
brown = brown / np.std(brown) * amplitude
```

---

### 1/f 噪声 (Fractional Noise)

**频谱特征**：**1/f^α 衰减**（α 可调，0 ≤ α ≤ 2）

| α 值 | 噪声类型 | 频谱斜率 |
|------|---------|---------|
| 0 | 白噪声 | 0 dB/倍频程 |
| 0.5 | 中间态 | -5 dB/倍频程 |
| 1.0 | 粉红噪声 | -10 dB/倍频程 |
| 1.5 | 中间态 | -15 dB/倍频程 |
| 2.0 | 布朗噪声 | -20 dB/倍频程 |

**用途**：可根据需要精细调节背景噪声的频谱特性

### 数学关系总结

| 噪声 | 功率谱密度 | 生成方法 |
|------|-----------|---------|
| 白噪声 | $P(f) = C$ | `randn()` |
| 粉红噪声 | $P(f) = C/f$ | `cumsum(randn())` |
| 布朗噪声 | $P(f) = C/f^2$ | `cumsum(cumsum(randn()))` |
| 1/f^α | $P(f) = C/f^\alpha$ | 分数积分滤波 |

**关键理解**：积分操作 = 低通滤波 = 增强低频 = 产生 1/f 特性

## 相关概念

### SNR（信噪比）

```
SNR (dB) = 20 × log10(信号幅度 / 噪声幅度)
```

- **高SNR（>20dB）**：信号清晰，易于分析
- **中SNR（10-20dB）**：需要滤波处理
- **低SNR（<10dB）**：信号难以提取

### 实际EEG的噪声来源

1. **生理性噪声**：
   - 眼动（EOG）： frontal区域，大振幅
   - 心跳（ECG）：有节律的脉冲
   - 肌肉（EMG）：高频成分

2. **环境噪声**：
   - 电源线干扰：50/60Hz
   - 电磁干扰：变压器、电机等

3. **设备噪声**：
   - 放大器噪声：电子热噪声
   - 电极噪声：接触不良、极化
