## 快速开始

### 标准仿真流程

```
1. 源空间配置 (Source Config)
   └─ 加载 -src.fif 文件
   
2. Patch管理 (Patch Manager)
   └─ 创建Patch，设置波形参数
   
3. 耦合配置 (Coupling Manager) [可选]
   └─ 设置Patch间连接
   
4. 电极通道 (Electrode Channels)
   └─ 选择10-20布局，选中通道
   
5. 噪声添加 (Noise Manager) [可选]
   └─ 添加背景噪声
   
6. 输出设置 (Output Page)
   └─ 设置采样率和输出文件
   
7. 开始仿真
   └─ 在Signal Page监控波形
```

## 核心概念

### 1. 源空间 (Source Space)

大脑皮层表面的离散化表示，包含数千个源点（vertices）。是仿真的空间基础。

### 2. Patch

空间上相邻的偶极子组，共享相同的波形设置。简化大规模仿真管理。

### 3. 偶极子 (Dipole)

电流源模型，代表一个神经元群的总和电流。每个偶极子有位置和方向。

### 4. 前向模型 (Forward Model)

描述从源（大脑）到传感器（头皮电极）的信号传导物理过程。

### 5. 耦合 (Coupling)

不同脑区间的功能连接，模拟神经信号的传播和相互作用。

## 常用参数参考

### 采样率选择

| 研究类型 | 推荐采样率 | 说明 |
|----------|-----------|------|
| 慢波/睡眠 | 200-250 Hz | 满足Nyquist，数据量小 |
| 标准EEG | 500-1000 Hz | 常规分析 |
| ERP研究 | 1000-2000 Hz | 精确潜伏期测量 |
| 高频Gamma | 2000-5000 Hz | >100Hz信号 |

### 典型信号幅度

| 信号类型 | 幅度范围 | 说明 |
|----------|----------|------|
| 背景EEG | 10-50 μV | 正常脑电活动 |
| ERP成分 | 5-20 μV | 事件相关电位 |
| 强反应 | 50-100 μV | 明显响应 |
| 癫痫样 | >100 μV | 病理活动 |

### 滤波参数

| 应用场景 | 高通 | 低通 | 陷波 |
|----------|------|------|------|
| 原始信号 | - | - | - |
| ERP分析 | 0.1 Hz | 30 Hz | 50 Hz |
| 频谱分析 | 0.5 Hz | 100 Hz | 50 Hz |
| 慢波分析 | - | 4 Hz | - |

## 故障排除速查

| 问题 | 检查项 | 解决方案 |
|------|--------|----------|
| 无信号显示 | 通道是否选中 | 在Channels页面选择通道 |
| 波形异常 | 滤波参数 | 调整Highpass/Lowpass |
| 基线漂移 | 高通设置 | 设置Highpass为0.5-1Hz |
| 50Hz干扰 | 陷波滤波 | 启用Notch Filter |
| 仿真卡顿 | 采样率/通道数 | 降低采样率或减少通道 |
| 数据未保存 | 输出路径 | 检查路径和权限 |

## 文件格式说明

### 输入文件

| 扩展名 | 内容 | 来源 |
|--------|------|------|
| `-src.fif` | 源空间 | MNE/FreeSurfer |
| `-fwd.fif` | 前向模型 | MNE计算 |
| `.fif` | EEG数据 | MNE采集 |

### 输出文件

| 扩展名 | 用途 | 软件支持 |
|--------|------|----------|
| `.fif` | MNE原生 | MNE-Python |
| `.edf` | 国际标准 | 大多数EEG软件 |
| `.csv` | 通用文本 | Excel/Matlab |
| `.npy` | Python数组 | NumPy |
