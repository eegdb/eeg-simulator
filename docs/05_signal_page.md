# 实时信号页面 (Signal Page)

## 功能概述

实时信号页面是仿真器的核心监控界面，实时显示各通道的EEG波形、频谱分析和地形图。用户可以在此调整滤波参数、选择显示通道、监控仿真状态。

## 界面布局

- **上方控制栏**：滤波参数和时间窗口设置
- **左侧主区域**：实时波形图
- **右侧**：FFT频谱 + 热力图/地形图

## 参数详解

### 1. 滤波参数 (Filter Settings)

#### 1.1 高通滤波 (Highpass)

**参数**：`Highpass` (单位：Hz)

**范围**：0 - 100 Hz

**默认值**：0.5 Hz

**功能**：抑制低频成分

**物理意义**：
- 去除基线漂移
- 去除皮肤电位缓慢变化
- 去除运动伪迹的低频成分

**推荐设置**：
| 应用场景 | 高通频率 |
|----------|----------|
| ERP分析 | 0.1-0.5 Hz |
| 频谱分析 | 0.5-2 Hz |
| 高频振荡 | 4-8 Hz |

---

#### 1.2 低通滤波 (Lowpass)

**参数**：`Lowpass` (单位：Hz)

**范围**：0 - 500 Hz

**默认值**：100 Hz

**功能**：抑制高频成分

**物理意义**：
- 去除高频噪声
- 抗混叠（如果后续降采样）
- 限制分析频带

**推荐设置**：
| 应用场景 | 低通频率 |
|----------|----------|
| 慢波分析 | 4-8 Hz |
| ERP分析 | 30-50 Hz |
| 全频带 | 100-200 Hz |
| Gamma分析 | 100-150 Hz |

---

#### 1.3 工频陷波 (Notch Filter)

**参数**：`Notch (50Hz)`

**类型**：复选框

**默认值**：启用

**功能**：精确去除50Hz（中国/欧洲）或60Hz（美国）工频干扰

**原理**：
- 使用IIR陷波滤波器
- Q值决定带宽（窄带去除）
- 不影响邻近频率成分

**Q值设置**（在Settings中）：
- 高Q（>100）：非常窄，可能不稳定
- 中Q（30-50）：推荐，平衡效果和稳定性
- 低Q（<20）：较宽，可能影响邻近频率

---

### 2. 时间窗口 (Time Window)

**参数**：`Time Window` (单位：秒)

**范围**：1 - 60 秒

**默认值**：10 秒

**功能**：显示的历史数据时长

**意义**：
- 短时间窗口（1-5s）：观察瞬态变化
- 中等窗口（10s）：常规监测
- 长时间窗口（30-60s）：趋势分析

**性能考虑**：
- 时间窗口越长，显示的数据点越多
- 30fps刷新率下，10秒窗口 = 10,000点
- 过长的窗口可能影响实时性能

---

### 3. FFT频谱分析

**显示**：右侧上方图表

**X轴**：频率（Hz）

**Y轴**：功率（μV²/Hz 或 dB）

**功能**：
- 实时显示选中通道的功率谱
- 识别主要频率成分
- 监测频带功率变化

**频带标记**：
| 频带 | 频率范围 | 颜色 | 典型活动 |
|------|----------|------|----------|
| Delta | 0.5-4 Hz | 红色 | 深睡、婴儿 |
| Theta | 4-8 Hz | 橙色 | 困倦、记忆 |
| Alpha | 8-13 Hz | 黄色 | 放松、闭眼 |
| Beta | 13-30 Hz | 绿色 | 专注、运动 |
| Gamma | 30-100 Hz | 蓝色 | 认知、注意 |

**通道选择**：使用FFT下方的下拉菜单选择分析通道

---

### 4. 热力图/地形图 (Heatmap)

**显示**：右侧下方图表

**类型**：
- **电极图**：显示电极位置和活动强度
- **地形图**：2D头皮表面插值显示

**颜色编码**：
- 红色/黄色：高活动（正电位）
- 蓝色/紫色：低活动（负电位）
- 绿色/黑色：基线水平

**刷新间隔**：在Settings中设置（默认1000ms = 1秒）

**用途**：
- 观察空间分布模式
- 识别源位置
- 可视化传播模式

---

## 波形显示区域

### 通道显示

**每通道显示内容**：
- 通道名称（左上角）
- 波形曲线
- Y轴范围（自动调整）

**交互**：
- **滚轮缩放**：放大/缩小Y轴
- **双击重置**：恢复默认视图

### Y轴范围

自动根据信号幅度调整：
- 典型EEG：±50 μV
- 高幅ERP：±100 μV
- 含噪声信号：±200 μV

## 使用流程

### 常规监测

```
1. 开始仿真后，自动显示波形
   ↓
2. 在左侧观察各通道波形
   ↓
3. 在右侧观察FFT频谱
   ↓
4. 在右下角观察空间分布
   ↓
5. 根据需要调整滤波参数
```

### 调整滤波

```
1. 观察原始波形（关闭滤波观察噪声）
   ↓
2. 逐步调整Highpass（去除漂移）
   ↓
3. 逐步调整Lowpass（平滑波形）
   ↓
4. 启用Notch（去除工频）
   ↓
5. 观察FFT确认频带干净
```

## 数据分析

### 1. 频带功率监测

在FFT图中观察各频带功率：
- Alpha波：闭眼时在枕叶增强
- Beta波：运动时在对侧运动区增强
- Theta波：困倦时增强

### 2. 相位同步分析

观察不同通道波形的时间关系：
- 同相：同时达到峰值
- 反相：峰值与谷值对齐
- 延迟：有时间差

### 3. 空间模式识别

在热力图中识别典型模式：
- **局灶性**：单个热点
- **广泛性**：全头皮分布
- **侧向性**：左右不对称

## 故障排除

| 问题 | 可能原因 | 解决方案 |
|------|----------|----------|
| 波形剧烈跳动 | 未滤波/低SNR | 启用滤波，检查噪声设置 |
| 基线漂移 | 未启用高通 | Highpass设为0.5-1Hz |
| 50Hz条纹 | 工频干扰 | 启用Notch滤波 |
| 所有通道相同 | 仿真源单一 | 增加更多Patch |
| 无信号显示 | 通道未选中 | 在Channels页面选择通道 |

---

## 相关概念

### 采样率与显示

- 采样率1000Hz = 每秒1000个数据点
- 10秒窗口 = 10,000个显示点
- 30fps刷新 = 每33ms更新一帧

### 滤波器延迟

实时滤波引入的延迟：
- 高通/低通：约 阶数/频率 秒
- 例如：4阶@1Hz ≈ 4秒群延迟

注意：这是显示延迟，不影响保存的数据

### 频谱泄漏

FFT的固有限制：
- 有限时间窗口导致频率模糊
- 使用Hanning窗减少泄漏（已实现）
