# 耦合管理器 (Coupling Manager)

## 功能概述

耦合管理器用于定义和管理不同Patch之间的功能连接（耦合关系）。通过设置源Patch对目标Patch的影响，可以模拟真实大脑中的功能网络连接，如信息传递、神经振荡同步等。

## 界面布局

- **上方**：耦合模型列表
- **中间**：源Patch和目标Patch选择
- **下方**：耦合参数配置

## 耦合类型详解

### 1. 单向耦合 (Unidirectional)

**描述**：信号从源Patch单向传递到目标Patch

**应用场景**：
- 感觉通路传递（如视觉皮层→顶叶）
- 前馈连接
- 信息流追踪

**参数**：

| 参数 | 单位 | 默认值 | 说明 |
|------|------|--------|------|
| Source | - | - | 源Patch（驱动方） |
| Target | - | - | 目标Patch（响应方） |
| Strength | 0-1 | 0.5 | 耦合强度 |
| Delay | ms | 10 | 传递延迟 |

---

### 2. 双向耦合 (Bidirectional)

**描述**：两个Patch相互影响

**应用场景**：
- 功能网络同步
- 双向通信（如左右半球间）
- 共振增强

**参数**：

| 参数 | 单位 | 默认值 | 说明 |
|------|------|--------|------|
| Patch A | - | - | 第一个Patch |
| Patch B | - | - | 第二个Patch |
| Strength | 0-1 | 0.5 | 耦合强度 |
| Delay | ms | 10 | 传递延迟 |

---

### 3. 延迟耦合 (Delayed Coupling)

**描述**：考虑神经传导延迟的耦合

**应用场景**：
- 远距离脑区连接（需要考虑传导时间）
- 长程纤维束传导
- 精确时序研究

**参数**：

| 参数 | 单位 | 默认值 | 说明 |
|------|------|--------|------|
| Source | - | - | 源Patch |
| Target | - | - | 目标Patch |
| Strength | 0-1 | 0.5 | 耦合强度 |
| Delay | ms | 20-100 | 长距离延迟 |

**延迟参考值**：

| 连接类型 | 典型延迟 |
|----------|----------|
| 局部连接 | 1-10 ms |
| 同半球内 | 10-30 ms |
| 跨半球 | 30-80 ms |
| 长程纤维 | 50-150 ms |

---

## 参数详解

### 1. 耦合强度 (Strength)

**范围**：0.0 - 1.0

**默认值**：0.5

**物理意义**：
- 源信号对目标信号的贡献比例
- 0 = 无耦合
- 1 = 强耦合（源信号完全主导）

**建议值**：
- 弱耦合：0.1-0.3（微弱影响）
- 中等耦合：0.3-0.6（明显影响）
- 强耦合：0.6-0.9（主导作用）

**计算公式**：
```
目标信号 = 目标原始信号 + Strength × 延迟后的源信号
```

---

### 2. 延迟 (Delay)

**单位**：毫秒 (ms)

**默认值**：10 ms

**物理意义**：
- 神经信号在轴突上的传导时间
- 突触传递延迟
- 距离越远，延迟越大

**计算方法**：
```
延迟 = 距离 / 传导速度 + 突触延迟
```

- 传导速度：1-10 m/s（无髓鞘纤维）
- 传导速度：20-100 m/s（有髓鞘纤维）
- 突触延迟：1-3 ms

---

### 3. 耦合模式 (Coupling Mode)

**线性耦合 (Linear)**
```
输出 = 输入 × Strength
```
简单直接，适用于大多数情况

**非线性耦合 (Nonlinear)**
```
输出 = tanh(输入 × Strength)
```
考虑神经元饱和效应

**相位耦合 (Phase)**
只影响振荡相位，不影响幅度
用于研究振荡同步

## 使用流程

### 创建耦合

```
1. 点击"Add Coupling"
   ↓
2. 选择Source Patch（源）
   ↓
3. 选择Target Patch（目标）
   ↓
4. 选择Coupling Type（类型）
   ↓
5. 设置Strength（强度）
   ↓
6. 设置Delay（延迟）
   ↓
7. 点击"OK"保存
```

### 示例：视觉-运动通路

```
Patch: V1（初级视觉皮层）
  ↓  Strength=0.6, Delay=15ms
Patch: MT（运动视觉区）
  ↓  Strength=0.5, Delay=20ms  
Patch: M1（初级运动皮层）
```

模拟视觉引导的运动反应

## 典型网络配置

### 1. 感觉-认知-运动通路

```
感觉皮层
   ↓ (30ms)
联合皮层
   ↓ (40ms)
前额叶（决策）
   ↓ (50ms)
运动皮层
```

**用途**：模拟简单反应任务

---

### 2. 默认模式网络 (DMN)

```
后扣带回(PCC) ←→ 内侧前额叶(mPFC)
       ↑                  ↑
   角回 ←→ 海马          ↑
```

**特点**：
- 强双向耦合
- 低频率振荡（<10Hz）
- 静息态活跃

**用途**：模拟静息态脑活动

---

### 3. 左右半球同步

```
左半球Patch A ←→ 右半球Patch A'
左半球Patch B ←→ 右半球Patch B'
```

**延迟**：20-80ms（胼胝体传导）

**用途**：研究半球间信息整合

## 注意事项

### 1. 避免循环依赖

不建议创建闭合环路：
```
A → B → C → A
```
可能导致数值不稳定

### 2. 总强度控制

一个Patch接收的所有耦合强度之和：
```
Σ(输入耦合强度) < 2.0
```
超过此值可能导致信号发散

### 3. 延迟与采样率

确保延迟能被采样率整除：
```
Delay ≥ 1 / Sampling_Rate
```
例如：1000Hz采样率，最小延迟1ms

## 相关概念

### 功能连接 vs 结构连接

| 特性 | 功能连接 | 结构连接 |
|------|----------|----------|
| 定义 | 统计相关性 | 解剖纤维 |
| 方向性 | 可以是双向 | 有方向性 |
| 延迟 | 隐含 | 明确 |
| 测量 | fMRI/EEG相干性 | DTI/fiber tracing |

本仿真器实现的是**功能连接**，但考虑了结构连接的方向性和延迟。

### 相干性 (Coherence)

衡量两个信号的频率同步性：
- 高相干性：耦合强，活动同步
- 低相干性：独立活动

### Granger因果

统计推断信息流方向：
- 如果A能预测B，则A→B
- 注意：统计因果≠生理因果
