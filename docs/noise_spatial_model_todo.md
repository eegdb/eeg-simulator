## 背景

在单 Patch、无耦合、MNE 前向投影的测试中，Fp1/Fp2/F7 的**源信号**在峰时刻是对齐的（Fp7 可能反相 180°，属 lead field 正常现象）。  
开启噪声后，各通道波形出现** apparent 相位偏移**、相关度下降——根因是当前**每通道独立生成噪声**，而非源投影问题。

## 当前实现（as-is）

### 代码位置

| 模块 | 文件 | 说明 |
|------|------|------|
| 噪声叠加 | `eeg_simulator/core/simulator/signal.py` → `_add_noise_batch()` | 按通道循环，分别生成并叠加噪声 |
| 连续噪声生成 | `eeg_simulator/core/signal_engine.py` → `generate_continuous_noise()` | 白/粉/棕/EOG/ECG/工频等 |
| 状态存储 | `EEGSimulator._noise_states` | `{channel: {noise_type: state_dict}}`，**通道间不共享** |
| UI 配置 | `eeg_simulator/ui/dialogs/noise_manager.py` | 噪声类型与幅度参数 |
| 项目持久化 | `meta.json` → `"noise": [...]` | 噪声配置列表 |

### 叠加公式

```
eeg_data[ch] = 投影信号[ch] + Σ noise_type( 独立生成(ch) ) × 1e-6   # 噪声 μV → V
```

每种噪声类型、每个通道维护**独立的随机状态**，保证时间连续，但**通道之间无空间相关**。

### 当前行为的特点

| 优点 | 缺点 |
|------|------|
| 实现简单，实时流式友好 | 与真实 EEG 空间统计不符 |
| 时间连续（无块边界断裂） | EOG/ECG/工频等伪迹应高度相关，现被独立化 |
| 可用于算法「最坏情况」压测 | 易掩盖源信号通道间对齐关系 |
|  | 热力图/多通道对比时各通道显得「各自为政」 |

## 真实 EEG 中的空间特性（参考）

| 噪声类型 | 真实空间行为 | 当前独立 per-channel 是否合理 |
|----------|--------------|------------------------------|
| **工频 50/60 Hz** | 全通道高度同相，幅度略有差异 | ❌ 应共享单一源 |
| **EOG（眼电）** | 近似全局偶极子场，额区强、枕区弱 | ❌ 应公共源 + 空间权重 |
| **ECG（心电）** | 全头皮近似同相 | ❌ 应共享模板 |
| **EMG（肌电）** | 局部电极簇相关 | ⚠️ 应局部相关，非完全独立 |
| **白/粉/棕背景** | 公共模式 + 通道独有成分混合 | ⚠️ 完全独立略高估去相关 |

业界常见做法：使用**噪声协方差矩阵**（非对角）或「公共成分 × 空间拓扑 + 独立残差」。

---

## 已观察到的用户侧影响

1. **波形像有相位偏移**：实为各通道独立噪声叠加后，峰时刻被随机成分顶歪。  
2. **关闭噪声后**通道对齐关系恢复，可验证源/投影链路正常。  
3. **滤波**：噪声在投影后、滤波前叠加；各通道滤波状态独立，会进一步放大通道间差异。

## 目标模型（to-be，待后续实现）

### 推荐结构

```
通道噪声[ch] = Σ_type (
    公共噪声_type(t) × 空间权重_type[ch]
    + 独立噪声_type[ch](t) × (1 - 混合比)
)
```

- **公共噪声**：每种类型只生成一次时间序列（全局状态）。  
- **空间权重**：由电极位置 / 前向 lead field / 预设拓扑（额区 EOG 大等）得到。  
- **独立残差**：保留少量 per-channel 成分，模拟放大器差异。  
- **混合比**：可配置，默认因类型而异（见下表）。

### 分类型建议

| 类型 | 公共成分占比 | 空间权重来源 | 优先级 |
|------|-------------|-------------|--------|
| `line` 工频 | ~100% | 全通道等幅 + 小幅随机 | P0 |
| `eog` | ~90% | 额区电极权重大（可用简单 10-20 距离或 lead field） | P0 |
| `ecg` | ~95% | 全通道同相 | P1 |
| `emg` | ~70% | 颞/额局部权重 | P2 |
| `white` / `pink` / `brown` | ~50–70% | 可选协方差矩阵或 lead field 投影 | P2 |

### 状态存储（草案）

```python
# 全局（公共源）
_noise_states_global: {noise_type: state_dict}

# 通道（仅独立残差）
_noise_states_independent: {channel: {noise_type: state_dict}}
```
