# 电极通道页面 (Electrode Channels Page)

## 功能概述

电极通道页面用于配置仿真中使用的EEG电极通道。用户可以加载标准电极布局（如10-20系统）、自定义布局，或从实际EEG数据中提取通道配置。

## 界面布局

- **上方**：电极布局和通道选择控制
- **左侧**：电极列表（带选择框）
- **右侧**：电极位置可视化图

## 参数详解

### 1. 电极布局选择

#### 1.1 标准布局

**参数**：`Electrode Layout`

**选项**：
| 布局名称 | 通道数 | 用途 |
|----------|--------|------|
| 10-20 System | 21 | 国际标准临床布局 |
| 10-10 System | 64+ | 高密度科研布局 |
| 10-05 System | 128+ | 超高密度研究 |
| MEG Channels | 306 | MEG传感器（如果可用） |

**10-20系统标准通道**：
```
前额：Fp1, Fp2, Fpz
额叶：F7, F3, Fz, F4, F8
中央：T7, C3, Cz, C4, T8
顶叶：P7, P3, Pz, P4, P8
枕叶：O1, Oz, O2
参考：A1, A2（耳垂）
```

---

#### 1.2 从文件加载

**按钮**：`Load Custom Layout`

**支持格式**：
- `.csv`：逗号分隔，格式：`通道名, X, Y, Z`
- `.txt`：空格或制表分隔
- `.loc/.locs`：EEGLAB格式

**坐标系统**：
- 使用球坐标或笛卡尔坐标
- 原点在头顶中心
- X轴：左右（右为正）
- Y轴：前后（前为正）
- Z轴：上下（上为正）

---

### 2. 通道选择

#### 2.1 全选/全不选

**按钮**：
- `Select All`：选择所有通道
- `Clear All`：取消所有选择

#### 2.2 区域选择

**功能**：按脑区批量选择通道

**区域**：
| 区域 | 包含通道 |
|------|----------|
| Frontal | Fp1, Fp2, Fz, F3, F4, F7, F8 |
| Central | Cz, C3, C4, T7, T8 |
| Parietal | Pz, P3, P4, P7, P8 |
| Occipital | O1, O2, Oz |
| Temporal | T7, T8, P7, P8 |

---

### 3. 电极可视化

#### 3.1 显示选项

**复选框**：
- `Show Labels`：显示电极标签
- `Show Heatmap`：叠加活动热力图

#### 3.2 视图模式

**选项**：
- `Top View`：俯视图（从上往下看）
- `Back View`：后视图（从后往前看）
- `3D View`：三维旋转视图（如果支持）

---

### 4. 通道映射

#### 4.1 标准命名映射

**功能**：将标准10-20命名映射到MNE前向模型的通道名

**示例**：
```
EEG标准名 → MNE内部名
Fp1 → EEG 001
Fp2 → EEG 002
Cz  → EEG 064
```

**意义**：
- MNE前向模型使用内部编号
- 映射确保正确投影到对应电极
- 自动完成，无需手动设置

#### 4.2 自定义映射

**功能**：手动编辑通道名称映射

**用途**：
- 非标准电极系统
- 实际采集数据通道重命名
- 多会话数据统一

## 使用流程

### 标准仿真

```
1. 在Layout下拉菜单选择"10-20 System"
   ↓
2. 等待加载标准电极位置
   ↓
3. 在列表中选择需要的通道（或使用Select All）
   ↓
4. 确认右侧视图中电极位置正确
   ↓
5. 切换到Signal页面查看波形
```

### 自定义布局

```
1. 准备布局文件（CSV格式）
   格式：通道名, X, Y, Z
   ↓
2. 点击"Load Custom Layout"
   ↓
3. 选择文件并确认加载
   ↓
4. 检查可视化位置是否正确
   ↓
5. 保存布局（可选）
```

## 布局文件格式

### CSV格式示例

```csv
# 电极布局文件
# 格式: 通道名, X, Y, Z (单位: cm)
Fp1, -7.07, 7.07, 0.0
Fp2, 7.07, 7.07, 0.0
F7, -8.66, 5.0, 0.0
F3, -4.0, 6.93, 5.0
Fz, 0.0, 7.0, 5.0
F4, 4.0, 6.93, 5.0
F8, 8.66, 5.0, 0.0
...
```

### 坐标说明

- **单位**：厘米（cm）或归一化（0-1）
- **原点**：头顶中心（Cz上方）
- **X轴**：左右方向，右为正
- **Y轴**：前后方向，前为正
- **Z轴**：上下方向，上为正

## 注意事项

### 1. 通道数与性能

- 更多通道 = 更多计算
- 21通道（10-20）：轻松实时
- 64通道（10-10）：良好性能
- 128+通道：可能降低帧率

### 2. 参考电极

- 标准10-20使用耳垂参考（A1, A2）
- 本仿真器使用平均参考
- 选择通道时应包含足够覆盖

### 3. 与源投影匹配

确保通道位置与源投影一致：
- 使用与MNE前向模型对应的布局
- 自定义布局需要重新计算前向模型

---

## 故障排除

| 问题 | 原因 | 解决 |
|------|------|------|
| 通道显示在错误位置 | 坐标单位错误 | 检查坐标单位（cm vs m） |
| 布局文件加载失败 | 格式错误 | 检查CSV格式和分隔符 |
| 某些通道无信号 | 通道未选中 | 检查复选框状态 |
| 热力图位置偏移 | 布局与Montage不匹配 | 使用匹配的Montage |

## 相关概念

### 10-20命名规则

```
F - Frontal (额叶)
T - Temporal (颞叶)
C - Central (中央)
P - Parietal (顶叶)
O - Occipital (枕叶)

z - 中线 (zero)
even - 右半球
odd - 左半球

数字表示与中心的距离：
- 1,2: 最近
- 3,4: 中等
- 5,6,7,8: 最远
```

### 参考电极

- **单极记录**：活动电极 vs 参考电极
- **平均参考**：所有电极的平均作为参考
- **双极记录**：两个活动电极之间的差值

本仿真器使用平均参考系统。
